# Kingdom Simulator

A C# console application demonstrating advanced Object-Oriented Programming (OOP) concepts. Developed as a major semester project at ELTE (Eötvös Loránd University).

# Tech Stack & Highlights
- Language/Framework: C# (.NET)
- Architecture: Strict OOP principles and Design Patterns (State Pattern, Visitor / Double Dispatch)
- Testing: 100% Code Coverage using MSTest, following the AAA (Arrange-Act-Assert) pattern

# How to Run
1. Clone the repository and open the `.sln` file in Visual Studio.
2. Build the solution to restore dependencies.
3. Run the application to start the simulation.
4. Optional: Open the Test Explorer to execute the unit test suite.
